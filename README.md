# OJT Progress Tracker

A simple mini productivity web application built using HTML, CSS, and JavaScript.

## Features
- Add daily learning tasks
- Set task date, category and status
- Update task status:
  - Pending
  - In Progress
  - Complete
- Store data using Local Storage

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
