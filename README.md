# OJT Progress Tracker

A simple productivity web application developed using HTML, CSS, and JavaScript to manage and track daily OJT (On-the-Job Training) tasks.

## Project Overview

The OJT Progress Tracker helps students maintain a record of their daily learning activities, monitor task progress, and store important notes such as learnings, errors, and mentor feedback.

All data is stored in the browser using Local Storage, allowing tasks to remain available even after refreshing the page.

---

## Features

### Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Search tasks instantly

### Progress Tracking
- Mark tasks as:
  - Pending
  - In Progress
  - Complete
- Dynamic progress bar
- Automatic progress calculation

### Summary Dashboard
- Total Tasks
- Completed Tasks
- Pending Tasks

### Technologies Selection
Multiple technologies can be selected for a single task:
- HTML
- CSS
- JavaScript
- Java
- Git
- GitHub

### Notes Section
Store:
- Errors faced
- Learnings
- Mentor feedback

### Local Storage
- Data is saved in browser Local Storage
- Tasks remain available after page refresh
- No database required
